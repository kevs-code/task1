Earthquakes API Description

Select country to use CountryInfo API to acquire bounding box parameters for Earthquakes API.

Result : returns a list of earthquakes, ordered by magnitude (default size = 10 rows).
Cities API Description

Select country to use CountryInfo API to acquire bounding box parameters for Cities API.

Result : returns a list of cities and placenames, ordered by relevancy (capital/population, default size = 10 rows).
findNearbyWeather API Description

Select country to use CountryInfo API to acquire bounding box parameters for Cities API to acquire long and lat for findNearByWeather API.

Result : weather station observations, to the first city in bounding box ordered by relevancy (capital/population).

Note : bounding boxes overlap, very simple first element implementation here e.g., selects Canadian capital for USA because of washington smaller population
